integer, parameter, private tfsf::dp = KIND(1.d0)

Incidenet Fields.

The Einc0, Einc1, and Hinc0 fields are the fields on the incident grid.

The EzInc, HxInc, and HyInc fields are used to update the actual FDTD fields.
